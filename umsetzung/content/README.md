# Content-Entwürfe → Veröffentlichung

Entwürfe hier entstehen aus der Content-Pipeline des Dashboards (Status „Entwurf“).
Ablauf bis „Veröffentlicht“:

1. **Review:** Fakten/Zahlen prüfen (Pflegebeträge ändern sich jährlich!), Platzhalter
   `[GROSSBUCHSTABEN]` mit echten Angaben füllen — nie erfundene Kundenstimmen/Zahlen.
2. **Wix einpflegen:** Ratgeber → Wix-Blog (Sitemap `sitemap.blog.xml` ist bereits aktiv);
   Landingpages → reguläre Seite. Meta-Title/-Description aus dem Entwurfskopf übernehmen.
3. **Schema einbauen** (FAQ-Block aus dem Entwurf, LocalBusiness aus `../schema/`).
4. **Interne Links** setzen (im Entwurfskopf bzw. -fuß aufgelistet).
5. Dashboard: Status auf **Veröffentlicht** + `published_url` eintragen.
6. Nach Indexierung: GSC-Abdeckung prüfen; Amplifikation („weißes Tier 2“):
   GBP-Beitrag + LinkedIn-Post auf den Artikel.
